# Cybersecurity Awareness Chatbot – Part 1

## Description
Console-based chatbot that educates users about password safety, phishing, and safe browsing.

## Features
- Voice greeting (WAV)
- Orbitz ASCII logo
- Personalised interaction
- Input validation
- Modular code (multiple classes)
- GitHub Actions CI

## How to Run
1. Open solution in Visual Studio 2022+
2. Ensure `greeting.wav` is present and set to "Copy Always"
3. Press F5
